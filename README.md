# Darth-Ference
proteomics_pipeline_145.R is the main pipeline for analyzing the dataset https://www.ebi.ac.uk/pride/archive/projects/PXD071481

proteinGroups_B.txt is a file we generated by processing the raw files of the dataset using MaxQuant. It was validated by comparison with the original proteinGroups from the dataset (compare_proteinGroupsAandB.R) 

sdrfFIX3.R is a script that generates sdrf_FIXED.tsv, which is required to resolve the mismatch in naming between the original SDRF file and our proteinGroups_B
